Este código crea una aplicación que muestra un JList con los elementos de una lista. La lista se representa como un objeto ObservableList, que se utiliza para notificar a los observadores cuando se agrega un elemento a la lista. El JFrame implementa el interfaz Observer y se registra como observador de la lista, por lo que recibirá notificaciones cada vez que se agrega un elemento a la lista. Cuando se notifica un cambio, se llama al método updateList para actualizar el JList con los elementos más recientes.
